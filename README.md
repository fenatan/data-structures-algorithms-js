## Installation

This is Node.js module available through npm.

First get yourself Node.js, and then
OS X & Linux:

```sh
$ npm install dsa-node --save
```

## Development setup

#### To install all development dependencies

```sh
$ npm install
```

#### To run tests

```sh
$ npm test
```
